自動化フレームワーク - ジョブ結果テスト

自動化フレームワークは、YAML ファイルで設定できる条件に基づいてジョブの結果をテストすることをサポートしています。

現在サポートされているテストは以下の通りです：

  • alert - アラートテスト - 指定されたアラートの有無を検証します
  • monitor - モニターテスト - 統計の閾値に基づいて長時間実行されるジョブを停止できるようにします
  • stats - 統計テスト - ZAPによって保持されている任意の統計をテストします
  • url - URLテスト - URLの有無、およびHTTPレスポンス/リクエストにおけるその特定の表現を検証します
