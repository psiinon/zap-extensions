自動化フレームワーク - GUI

自動化フレームワークには、現在開発中のGUIが備わっています。

自動化タブ

このタブでは、自動化ジョブの作成、ロード、編集、実行ができます。 このタブには2つのサブタブがあります。

ツールバーには以下のボタンがあります：

  • 新しいプラン… - 新しいプランダイアログを起動します
  • プランをロード… - YAMLファイルからプランをロードできます
  • プランを保存 - 現在のプランをYAMLファイルに保存します
  • プランを実行 - 現在のプランを実行します
  • プランを停止 - 現在のプランを停止します
  • ジョブを追加… - 現在のプランにジョブを追加するためのジョブの追加ダイアログを起動します
  • ジョブを削除… - 現在のプランから選択したジョブを削除します
  • ジョブを上へ移動 - 現在のプランで選択したジョブを1つ上に移動します
  • ジョブを下へ移動 - 現在のプランで選択したジョブを1つ下に移動します
  • テストを追加… - 現在のジョブにテストを追加するためのテストの追加ダイアログを起動します
  • テストを削除… - 現在のジョブから選択したテストを削除します

プランサブタブ

プランをグラフィカルに表示し、実行中はその状態も表示します。
プランの任意の要素をダブルクリックして編集できます。

また、プランテーブルから以下の一般的なジョブオプションを変更できます：

  • 有効 - 有効になっているジョブのみが実行されます。
  • 常に実行 - このオプションが有効なジョブは、プランが途中で終了した場合でも実行されます。

出力サブタブ

プランのロードまたは実行時に生成されたすべての出力が表示されます。
