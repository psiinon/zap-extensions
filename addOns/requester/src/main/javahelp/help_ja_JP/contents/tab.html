リクエスタータブ

リクエスタータブは、標準の「リクエスト」タブと「レスポンス」タブの隣にあるワークスペースパネルにあります。

リクエスタータブは、手動リクエストエディターダイアログのように動作しますが、必要なだけ多くのサブタブをサポートします。

他のZAPタブのリクエストを右クリックし、「リクエスタータブで開く...」を選択すると、そのリクエストを新しいリクエスタータブで開くことができます。 キーボードから Control / Command + "W" キーを使用しても同じことができます。

タブは最初は番号が振られていますが、タブをダブルクリックすると、名前を変更できるダイアログが表示されます。

各タブには、HTTPリクエストとレスポンス内の文字列を検索できるフッターがあります。 フッターには、大文字・小文字を区別しない一致の数が表示され、それらを順に移動できます。 フッターには、リクエストにかかった時間、ボディ長、合計長も表示されます。
