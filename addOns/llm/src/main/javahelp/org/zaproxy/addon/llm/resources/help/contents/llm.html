LLM Support

This optional add-on integrates LLM with ZAP. It allows you to configure ZAP to connect to an LLM of your choosing, and provides hooks for other add-ons to access the LLM.

ZAP will not share any data with LLMs unless you choose to install this add-on, and configure it to connect to a supported LLM.

All communications with LLMs will be shown in a sub-tab of the main Output tab.

This add-on provides the following functionality:

LLM Chat

An interactive chat interface that allows you to communicate directly with the configured LLM. You can ask questions, get explanations, and receive assistance in real-time. See the LLM Chat help page for more details.

Appending Data to Chat

Right-click menu items allow you to quickly append ZAP data to the LLM Chat input area:
  • Append Alert to LLM Chat: Available when right-clicking on alerts. Appends formatted alert information including risk, confidence, description, evidence, and URI. Supports appending multiple selected alerts.
  • Append Request to LLM Chat: Available when right-clicking on HTTP messages. Appends the HTTP request (headers and body) to the chat.
  • Append Response to LLM Chat: Available when right-clicking on HTTP messages. Appends the HTTP response (headers and body) to the chat.
  • Append Request & Response to LLM Chat: Available when right-clicking on HTTP messages. Appends both the request and response to the chat.
When data is appended, the chat panel automatically switches to the chat tab and positions the cursor at the end of the appended text, ready for you to add questions or send the message.
