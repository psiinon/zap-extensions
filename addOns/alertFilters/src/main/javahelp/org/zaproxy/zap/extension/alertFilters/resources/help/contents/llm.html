Alert Filter LLM Support

This add-on supports LLM integration. It will only be enabled if you have chosen to install the LLM add-on.

Alert Review

The add-on adds two right click menu items under AI Assisted that are enabled for a single alert or an alert tree:

Review and Update using LLM

Instructs the configured LLM to examine the alert(s) and determine the confidence level (False Positive, Low, Medium, or High) based on evidence from ZAP.
  • The confidence level of the alert(s) will be updated.
  • A new LLM Explanation will be added to the "Other Info" field.
  • The tag AI-Reviewed will be added to the alert.
  • Alerts that have already been reviewed (with the AI-Reviewed tag) are skipped.

Review and Report using LLM

Performs the same LLM review but does not update the alert. Instead, it provides a recommendation in the LLM Chat tab as to what the user should do. This is useful when you want an opinion without modifying the alert. Previously reviewed alerts are not skipped.
