クイックスタート

「クイックスタート」タブは、Webアプリケーションを迅速にテストするための簡単な方法を提供します。
画面上部には、以下の画面へ遷移するための3つのボタンが表示されます。

自動スキャン

「自動スキャン」画面では、アプリケーションに対して自動スキャンを開始できます。以下にURLを入力し、「攻撃」を押すだけです。
重要: テストの許可を明示的に取得したアプリケーションに対してのみ実行してください。

攻撃対象 URL

これは攻撃したいアプリケーションのURLで、「http://」または「https://」から始まります。
攻撃を続行するには、ZAPがこのURLにアクセスできる必要があります。

従来のスパイダーを使用

従来のスパイダーは、HTMLページ内のリンクを見つけることによってアプリケーションを探索します。 非常に高速ですが、JavaScriptを多用するアプリケーションには対応できません。

Ajaxスパイダーを使用

Ajaxスパイダーは、選択したブラウザを起動し、見つけたリンクをクリックすることでアプリケーションを探索します。 トラディショナルスパイダーよりも低速ですが、JavaScriptを適切に処理します。
このオプションは、Ajaxスパイダーアドオンがインストールされている場合にのみ表示されます。 利用できない場合は、ZAP マーケットプレイスから無料でダウンロードしてインストールできます。

利用可能なオプションは以下の通りです：
  • しない: 従来のスパイダーを使用する必要があります
  • モダンの場合: ZAPがターゲットをモダンアプリとして識別した際に実行されます - この場合は従来のスパイダーを使用してください
  • 常に: このオプションを選択した場合、従来のスパイダーを実行する必要はありません

手動探索

「手動探索」画面では選択したブラウザを起動でき、ZAP経由でプロキシしながらアプリケーションを探索できます。 プルダウンから使用したいブラウザを選択し、「ブラウザを起動」ボタンを押してください。
この方法で起動されたブラウザは、ZAP経由でプロキシするように設定され、証明書エラーを無視するため、 ZAPのルートCA証明書をインポートする必要はありません。
ZAPは、再起動時に前回選択したブラウザを記憶します。

ツールバーにもボタンが追加され、最後に選択されたブラウザが起動されます。アイコンは、該当するブラウザを示すように変化します。

注: ZAPがリッスンしているアドレスやポートを変更すると、起動されたブラウザは動作を停止します。

Seleniumアドオンがインストールされていない場合は、ブラウザをZAP経由でプロキシするように手動で設定するための手順が表示されます。

探索対象 URL

これは探索したいアプリケーションのURLで、「http://」または「https://」から始まります。
ZAPがこのURLにアクセスできる必要があります。

HUDを有効化

ZAP Heads Up Display (HUD) は、ZAPの主要機能をブラウザで利用できるようにします。 Webセキュリティ初心者に最適で、経験豊富なペンテスターがアプリケーションの機能に集中しながら、重要なセキュリティ情報と機能を活用することも可能にします。
このオプションは、HUDアドオンがインストールされている場合に有効になります。アドオンがインストールされていない場合は、ZAP マーケットプレイスから無料でダウンロードしてインストールできます。

詳細情報

「詳細情報」画面では、ZAPについてさらに学ぶためのローカルおよびオンラインリソースへのリンクが提供されます。

ニュース

3つのボタンの下には、ときどきニュース項目が表示されます。赤い「x」をクリックすると、現在のニュース項目が非表示になります。
